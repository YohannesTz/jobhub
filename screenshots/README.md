# JobHub Screenshots

This directory contains screenshots demonstrating the key features and interfaces of the JobHub application.

### Landing and Public Pages
- `landing-main.png` - Main landing page with hero section
- `joblistings.png` - Job listings page (accessible without login)
- `joblistings-detail.png` - Job detail page with full description
- `joblistings-detail-loginprompt.png` - Login prompt modal when applying without authentication

### Authentication
- `login.png` - Login page
- `creat-account.png` - Account creation form
- `talent-signup.png` - Talent (job seeker) registration

### Talent/User Features
- `talent-dashboard.png` - User dashboard showing applications

### Company Features

- `s3-ninja-dashboard.png` - S3 Ninja web interface for file storage management

## Key Features Demonstrated

### 1. Public Job Browsing
The `joblistings.png` and `joblistings-detail.png` screenshots show how users can browse and view job details without authentication, improving accessibility and SEO.

### 2. Login Prompt Modal
The `joblistings-detail-loginprompt.png` screenshot demonstrates the seamless login prompt that appears when unauthenticated users try to apply for jobs.

### 3. Complete User Journey
- Landing page introduction
- Job browsing
- Account creation
- Profile setup
- Job application

### 4. Company Workflow
- Company dashboard
- Company profile creation
- Job posting

### 5. File Storage
The S3 Ninja dashboard screenshot shows the S3-compatible storage system used for managing uploaded files like resumes and profile pictures.
